## 👀 ¿Cómo visualizar los proyectos?

### Proyectos con HTML, CSS y JavaScript

1. Ingresa a la carpeta del proyecto.
2. Descarga los archivos o clona el repositorio.
3. Abre el archivo `index.html` en tu navegador.

También puedes acceder al enlace de despliegue cuando esté disponible.

### Proyectos con PHP

Los proyectos que utilizan PHP necesitan ejecutarse mediante un servidor local o servicio de hosting compatible.

Puedes utilizar herramientas como:

- XAMPP
- Laragon
- WampServer
- Servidor integrado de PHP
- Render, Railway u otro servicio compatible

Ejemplo para ejecutar un proyecto con el servidor integrado de PHP:

```bash
php -S localhost:8000
```

Después, abre en el navegador:

```text
http://localhost:8000
```

## 🎯 Objetivo del repositorio

El objetivo de este repositorio es almacenar y compartir proyectos de desarrollo web para:

- Mostrar trabajos realizados.
- Practicar nuevas tecnologías.
- Compartir ejemplos con otros estudiantes.
- Facilitar el acceso al código fuente.
- Mantener un respaldo organizado de los proyectos.
