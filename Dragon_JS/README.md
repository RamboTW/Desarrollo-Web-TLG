# Vuelo del Dragón 🐉

Proyecto web que muestra una animación interactiva de un dragón volando, desarrollada con JavaScript y la librería Processing.js.

## Características

- Animación interactiva en un elemento `canvas`.
- Cambio de colores al tocar o hacer clic en la animación.
- Diseño responsive para teléfonos, tablets y computadoras.
- Opción de pantalla completa.
- Interfaz moderna y adaptable.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Processing.js 1.4.8

## Estructura del proyecto

```text
Sample/
├── index.html
├── sample.js
└── README.md
```

## Cómo ejecutar el proyecto 

- Abre la carpeta del proyecto en Visual Studio Code.
- Instala la extensión Live Server.
- Haz clic derecho sobre index.html.
- Selecciona Open with Live Server.
- Despues puede ejecutar el proyecto sin problemas con solo F5
- Se necesita conexión a internet para cargar la librería Processing.js desde el CDN.

## Uso

Haz clic o toca la animación para cambiar el color del dragón y del escenario.
